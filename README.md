# Job Market Analysis

Analyzed 60K+ job postings to uncover trends in hiring demand, salaries, remote work, locations, and in-demand skills using Python and SQL.

## Project Objective

The goal of this project is to explore the job market and answer questions such as:

* Which job titles are most in demand?
* Which companies hire the most?
* What salary trends exist across roles?
* Remote vs onsite job distribution
* Top locations for hiring
* Most requested technical skills

## Dataset

* Source: Public job postings dataset / simulated dataset
* Records analyzed: 60,000+
* Fields included:

  * Job Title
  * Company
  * Location
  * Salary
  * Work Type
  * Skills
  * Posting Date

## Key Insights

* Data Analyst, Data Scientist, and Business Analyst roles had highest demand.
* Remote jobs represented a significant share of listings.
* SQL, Python, Excel, and Power BI were among top requested skills.
* Major cities offered higher salaries and more openings.
